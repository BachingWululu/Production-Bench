AverageValueCalculation

Beschreibung

Dieser Funktionsbaustein dient zur fehlersicheren Mittelwertberechnung eines analogen Signals vom Datentyp Integer über eine benutzerdefinierte Dauer.

Funktionsweise

Aufbau des Bausteins LDrvSafe_AverageValueCalculation

Der Baustein errechnet den Durchschnittswert des Wertes welcher am Eingang "actValue" verschalten ist. Dieser wird über die über die Anzahl an Bausteindurchläufen gebildet.

Hinweis

Eine Wertänderung am Eingang „numberOfCycles“ wird erst wirksam, wenn der Baustein einen neuen Durchschnittswert "avgValue" berechnet und ausgegeben hat.

Sobald ein Durchschnittswert berechnet wurde, wird dieser am Ausgang "avgValue" bereitgestellt und das Bit "updatedValue" für einen Zyklus lang gesetzt.

Erkennt der Baustein einen Fehler, so wird der Ausgang "error" gesetzt, "updatedValue" nimmt den Wert "false" an und der Ausgang "avgValue" gibt den Wert "0" aus. Die Fehlerursache wird am Ausgang "diag" hinterlegt.

Der Ausgang "ackReq" wird gesetzt, wenn „error“ ansteht und alle Bedingungen, welche zum Fehler führten wieder behoben sind.

Eine Quittierung erfolgt durch positive Flanke am Quittiereingang "ack" und ist nur möglich, wenn „error“ gesetzt ist.

Schnittstellen

Eingangsparameter des Bausteins LDrvSafe_AverageValueCalculation

Parameter

Datentyp

Beschreibung

actValue

INT

Aktueller Wert

·   Wert aus dem der Mittelwert berechnet werden soll

·   Zulässiger Wertebereich:
0 ≤ „actValue“ ≤ 32767

numberOfCycles

INT

Zykluszahl

·   Anzahl der Bausteinaufrufe, über die der Mittelwert von "actValue" gebildet werden soll

·   Zulässiger Wertebereich:
2 ≤ „numberOfCycles“ ≤ 32767

·   Vorbelegung: "3"

ack

BOOL

Quittierung

·  Fehler am Baustein können mit einer steigenden Flanke quittiert werden

·  Quittierung ist nur möglich, wenn „error“ gesetzt ist und positive Flanke an "ack" ankommt

Ausgangsparameter des Bausteins LDrvSafe_AverageValueCalculation

Parameter

Datentyp

Beschreibung

avgValue

INT

Mittelwert

Zeigt den Mittelwert von "actValue" über die Anzahl der Aufrufe des Funktionsbausteins "numberOfCycles" an

updatedValue

BOOL

Mittelwert aktualisiert

Gibt an, ob ein neuer Mittelwert ansteht

error

BOOL

Fehler

Sobald ein Fehler im Baustein auftritt, wird dieses Bit gesetzt

ackReq

BOOL

Quittierung erforderlich

Zeigt an, wenn „error“ ansteht und alle Fehlerursachen welche zu diesem Fehler führten behoben sind

diag

WORD

Diagnose von Fehlern

Zeigt detaillierte Informationen zum Fehler an

Zusammensetzen des Diagnosewortes LDrvSafe_AverageValueCalculation

diag

Beschreibung

Bit 0

Unplausibler Wert

Wert "actValue" außerhalb des zulässigen Wertebereichs

Abhilfe

Wertebereich anpassen 0 ≤ "actValue" ≤ 32767

Bit 1

Unplausibler Wert

Wert "numberOfCycles" außerhalb des zulässigen Wertebereichs

Abhilfe

Wertebereich anpassen 2 ≤ " numberOfCycles" ≤ 32767

 

Hinweis

Auftreten mehrerer Fehler:
Bis zur Quittierung wird nur der zuerst aufgetretene Fehler im Diagnosewort angezeigt. Danach werden die Fehler im Diagnosewort nach ihrer Priorität angezeigt.
