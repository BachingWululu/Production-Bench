SinaGTlg30Control

Description

This function block is used to simply control the Safety Integrated Functions of SINAMICS G120 via PROFIsafe telegram 30.

Principle of operation

Structure of block LDrvSafe_SinaGTlg30Control

At the block input, the user interconnects Safety-related signals or sensors (e.g. Emergency Stop, guard door ...) with the Safety Integrated Functions.

The user must interconnect output "SinaGTlg30Control" with a variable of data type "LDrvSafe_typeSinaGTlg30Control".

Details on how to commission the block are provided in chapter General description.
