SinaTlg700Status

Description

This function block is used to simply evaluate the Safety Integrated Functions of SINAMICS via telegram 700.

Principle of operation

Structure of block LDrvSafe_SinaTlg700Status

 

The states of the Safety Integrated Functions of SINAMICS are represented at the output of the block.

Details on how to commission the block are provided in chapter Safety Info Channel (SIC).

 

The block is processed as long as the block input "enable" is set.

When an error occurs in the block, the output "error" is set; "status" shows the fault number. Errors are only acknowledged with a negative edge at input "enable".

Interface

Parameter

Data type

Description

BOOL

Activating the block with a rising edge. Acknowledging faults with a falling edge.
